# How 표현?
크게 2가지로 이해
Db상에서 쓰이는 SQL 구문, http 프로토콜 메서드로서의 이해
(Http Method)

* Create(Post)
SQL Insert


* Read(Get)
SQL Select


* Update(PUT(전체)/PATCH(부분))
SQL Update


* Delete(Delete)
SQL Delete
